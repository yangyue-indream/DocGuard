## 🧩 四步工作流

| 步骤 | 名称 | 说明 |
|:--:|------|------|
| ① | **上传文档** | 拖拽 PDF/TXT，自动提取文本 |
| ② | **AI Autopilot 审查** | AI 自动分类条款、评估风险、生成建议 |
| ③ | **人工复核** | 逐条查看 AI 建议，采纳/驳回/修改 |
| ④ | **导出报告** | Markdown + JSON 双格式导出 |

## 🎤 Pitch Deck 要点（3分钟）

> **问题**：中小企业签合同，请不起律师审查，只能硬着头皮签
>
> **方案**：DocGuard — AI Autopilot Agent，上传合同 3 秒出风险报告
>
> **为什么是 Autopilot**：AI 自动识别条款类型、评估风险、生成修改建议；人工复核关键决策
>
> **技术栈**：Qwen API + Streamlit + PyPDF2 + 结构化 Prompt 工程
>
> **市场**：中国每年合同超百亿份，中小企业法务需求巨大
>
> **为什么是我们**：极致简单（1 个文件），AI 审查质量不输初级律师，100% 开源
